# 内置相机 LUT 来源

- `sony-slog3-sgamut3cine-rec709.cube`：Sony 官方 S-Gamut3.Cine / S-Log3 软件处理 LUT（Cine+709）。
- `sony-slog2-sgamut-rec709.cube`：Sony 官方 S-Gamut / S-Log2 软件处理 LUT（Cine+709）。
- `panasonic-vlog-vgamut-v709.cube`：Panasonic 官方 V-Log / V-Gamut 转 V-709 LUT。

来源：

- https://www.sony.com/electronics/support/software/00263050
- https://www.sony.com/zh-cn/electronics/support/articles/00197011
- https://pro-av.panasonic.net/en/cinema_camera_varicam_eva/products/varicam_35/index.html

这些 LUT 仅在用户明确选择时应用到导出视频；“无”不会改变原始颜色。
